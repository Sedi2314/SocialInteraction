##📝 Project Overview:
This app is an app used for social interaction,where the user can type a time of day on the line below the text 'Enter time of day',once they enter the correct time of day they will be shown what to do in the orange block used for displaying what the user should do during that time of day,this app is handy as a person could forget what they were supposed to do during a specific time during the day.
##✨ Features:
Logic was powered using the kotline language to handle user data and conditional responses,also by using XML layouts to create an appealing app design for the user,this is a time based logic so you need to type a time of day for example 'morning' then once you press the submit  button response will be shown in the putput text view.
